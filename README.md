# StealthAnalytics

A player behavior analytics project built using Unreal Engine 5 and SQL.

## 🎮 Overview

This project is a small stealth prototype where player actions such as detection, mission success, and session timing are tracked and analyzed using SQL.

## 🛠️ Tech Stack

* Unreal Engine 5 (Blueprints)
* SQL (MySQL)
* CSV Data Logging

## 📊 Data Tracking

Events being tracked:

* SessionStart
* GuardDetectedPlayer
* MissionFailed
* MissionCompleted

## 🚧 Status

In progress — currently building gameplay and event tracking system.
